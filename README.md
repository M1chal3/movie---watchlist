# Movie Watchlist

A cinematic movie discovery app built with Next.js and TypeScript.

## Features
- Browse Popular, Top Rated and Upcoming movies
- Full movie detail page with trailer, cast and gallery
- Real-time data from TMDB API
- Updated as from the TMDB databse = up to date

## Tech Stack
Next.js, TypeScript, Tailwind CSS, TMDB API
